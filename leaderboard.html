Leaderboard (Coming Soon)

We are currently developing a leaderboard feature that will allow players to create usernames and compete against others on a daily basis.

Once accounts are introduced, players will be able to track their scores, compare results with others, and see how they rank each day.

The goal is to create a competitive but fair system where all players face the same daily challenge.

This feature is currently in development and will be introduced in a future update.
